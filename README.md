## hexo-theme-simple

一个简约但不简单的hexo主题

本主题首先配色是一股清新的性冷淡风格。希望能专注于文章，UI的呈现都是最为简洁的。

另外本主题有一个追求，就是访问快，压缩合并了js文件，对图片也做了懒加载处理。然而html和css的压缩处理，还是需要大家看看这篇[文章](http://hiluluke.cn/2016/08/19/hexo-compress/)。
